\# Sales Data Analysis



This project performs basic sales data analysis using \*\*Python\*\* and \*\*Pandas\*\*.

The dataset is loaded from a CSV file and analyzed to find key sales insights.



\## Technologies



\* Python

\* Pandas



\## Run the Project



Install pandas:



pip install pandas



Run the script:



python sales\_analysis.py
